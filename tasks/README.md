# SLAPR Tasks

Use this folder for task-based Codex work. Each task should be small enough to verify in one pass and should name the likely files, acceptance criteria, and commands to run.

## Task Template

```md
# Task: Short Title

## Goal

What user-visible or repo-quality result should exist when this is done?

## Scope

- Files or folders likely touched.
- Behavior that must not change.

## Acceptance Criteria

- Concrete checks a reviewer can verify.
- Required commands.

## Notes

- Risks, known tradeoffs, or follow-up tasks.
```
